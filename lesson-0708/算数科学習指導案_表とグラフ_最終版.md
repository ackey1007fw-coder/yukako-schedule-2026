# 算数科 学習指導案（最終版・7/8公開研）

日時：令和8年7月8日（水）5校時　学年・学級：第3学年1組　単元名：表とグラフ

来校者：麻布教育研究所 永島先生／市研修センター 十文字指導主事

---

## 1. 本時のねらい

道のりの資料を目的に応じて棒グラフに表し、目盛り・向き・並べ方を自分たちで工夫することを通して、グラフを根拠に判断・分類し、友達と聞き合いながら考えを深めることができる。

## 2. 本時で育てたい見方・考え方

「何を分かりやすくするためのグラフなのか」という目的から、目盛りの大きさ・縦横の向き・並べ方を選び、グラフの棒の長さや目盛りを根拠に説明する力。

## 3. 評価規準

| 観点 | 規準 | 評価場面 |
|:--|:--|:--|
| 知識・技能 | 資料の最大値に着目して目盛りを決め、棒グラフに表すことができる。 | 作図時のワークシート |
| 思考・判断・表現 | グラフを根拠に場所を分類し、理由を説明できる。 | グループ・全体での発言、根拠欄 |
| 主体的に学習に取り組む態度 | 友達のグラフや分け方と比べ、自分の考えを見直そうとしている。 | 振り返り記述 |

※ジャンプ課題は全員の到達を求めない。学級の3分の1が根拠をもって説明できれば十分とし、残りの児童は対話を通して近づく設計とする。

## 4. 本時の課題

### 共有課題（読むだけ・短く）

3年生の「いちばん好きなスポーツ」の表と棒グラフを見て、グラフにすると何が分かりやすいか説明しよう。

（既習の確認。作図はしない。水泳11人とドッジボール10人の小さな差、バスケとバレーの同数、「その他」の位置に軽く触れる程度で7分以内に切り上げる。）

### ジャンプ課題（描いて読む）

先生たちは、3年生の校外学習でどこへ見学に行くか迷っています。学校から7つの場所までの道のりを調べました。
先生たちが相談しやすいように、道のりが比べやすい棒グラフを作りましょう。
そして、グラフをもとに7つの場所を次の3つに分けて、なぜそう分けたのかをグラフを使って説明しましょう。
①見学先の候補にしやすい場所　②相談して決めたい場所　③歩いて行くには遠そうな場所

提示する表（意図的にバラバラの順で提示する）

| 場所 | 道のり |
|:--|--:|
| 翠ヶ丘公園 | 1700m |
| 須賀川郵便局 | 2300m |
| マイティウェルネスクラブ | 3100m |
| 須賀川牡丹園 | 800m |
| 須賀川警察署 | 1900m |
| 須賀川市中央図書館 tette | 1600m |
| 須賀川消防署 | 2600m |

配付物：作図用の大判方眼シート（目盛り数値なし・縦にも横にも使える）は**グループに1枚**。個人ワークシートの方眼欄は自分の案・下描き用とし、グループの1枚で練り合う（1人1枚だけだと無言の個人作業になりやすいため）。目盛り・向き・並べ方はすべて子どもに委ねる。

課題文と表は拡大掲示（または投影）で提示し、視写させない。

設計意図（子どもが考える必然性を埋め込んだ3点）

1. 目盛り：最大3100m。1目盛り100mでは31マス必要になり収まらない。「1目盛りを何mにするか」を考えざるを得ない。
2. 並べ方：表がバラバラなので、そのまま描く子と、長い順・短い順に並べ替える子が現れる。「どちらが比べやすいか」が対話になる。
3. 分類の境目：1600・1700・1900mと2300・2600mが接近しており、「②相談して決めたい場所」をめぐって判断が割れる。迷いは②という正式な分類として価値づけられる。

## 5. 本時の流れ

| 時間 | 学習活動 | 教師の関わり | ★我慢ポイント |
|:--|:--|:--|:--|
| 0〜3分 | 課題との出会い | 課題は拡大掲示で提示し、書かせる時間を取らない。めあては課題と出会ったあと、子どもの言葉を拾って短く確認する（先に板書して与えない） | ★冒頭の復習・説明を3分以内に |
| 3〜9分 | 共有課題（読む） | 「グラフだと何が分かりやすい？」→ペアで確認させる | ★教師がグラフのよさを説明しない |
| 9〜12分 | ジャンプ課題提示 | 表と方眼を配る。「先生たちは本当に迷っています」 | ★目盛り・向き・並べ方のヒントを一切言わない |
| 12〜27分 | グループで作図 | 「何を分かりやすくしたいグラフ？」「一番大きい数は何m？」 | ★「500mずつにすれば？」等の指示禁止。困りは記録だけ |
| 27〜32分 | グループで分類 | 「その分け方、グラフのどこを見ると分かる？」 | ★「これは②だね」と言わない |
| 32〜42分 | 全体共有 | 下記「取り上げる順番」参照 | ★子どもの発言を復唱しない。別の子に再生させる |
| 42〜45分 | 振り返り | 「今日、グラフのどこが根拠になった？」書けた子から数名、そのまま声に出して読む | ★教師の言葉でまとめ直さない・言い換えない |

## 6. 予想される児童の反応と教師の返し

| 予想される反応 | 返し |
|:--|:--|
| 「目盛り、何mにすればいいの？」 | もどす：「一番遠い場所は何m？ 方眼は何マスある？」 |
| 1目盛り100mで描き始めて入らなくなる | つなぐ：「同じことで困ってる人、グループにいない？」 |
| 表の順のまま描く／並べ替えて描くが混在 | つなぐ：「隣のグループのグラフと、並び方どこが違う？」 |
| 1900mを①にするか②にするかで割れる | 問い返す：「何と何を比べて迷ってるの？ それグラフのどこ？」 |
| 「先生、これで合ってる？」 | つなぐ：「先生に聞く前に、グループのみんなはどう言ってる？」 |
| 手が止まっている | もどす：「表のどこまで分かった？ どこで止まった？」 |
| 発言の強い子が結論を先に言う | つなぐ：「今の考え、違う人の言葉でもう一回聞きたいな。〇〇さん、どういうことか話してみて」 |
| 早く描き終えて手持ちぶさたになる | 揺さぶる：「目盛りを変えたら見え方はどう変わる？」「②を1つ①に動かすなら、グラフのどこを根拠にする？」グループ内で静かに根拠を確かめ合わせる |

## 7. グループ活動中の教師の動き（十文字先生の助言を反映）

- 指示を出した直後にすぐ机間巡視を始めず、いったん教室の後方や斜めから全体を見て、学びに混ざれていない子を見極めてから動く。
- 関わるのは「本当に困っている子・学びに混ざれない子」に絞る。教師に頼りがちな子には深入りせず、「グループで確認してみて」と友達につなぐ。
- 「わからない」と言えそうな子のわからなさを見つけておき、全体共有の起点として使う（わかっている子の正解発表から始めない）。その際、「全部わからない」ではなく「どこまで分かって、どこで止まったか」を本人の言葉で言えるよう促す。
- 早く終わったグループには新しい説明を与えず、グループ内で静かに根拠を確かめ合わせるか、上記の揺さぶりを返す。
- 目盛りの違うグラフ、並べ方の違うグラフを描いたグループを記録し、全体共有で対比させる。

## 8. 全体共有で取り上げる順番

1. グラフの見た目の違いから入る：並べ替えたグラフとそのままのグラフ、目盛りの違うグラフを黒板に並べ、「同じ表なのに、何が違う？」→グループに返す。
2. 「比べやすいのはどっち？ それはなぜ？」で、目的（先生たちの相談）に戻す。
3. 分類の境目へ：1900mが①か②かで割れた子を対比させ、根拠を言わせる。教師は復唱せず、「今の〇〇さんの考え、どういうことか隣の人に説明してみて」と再生・確認をグループに落とす。話者が1〜2人進むごとにグループへ落とし、聞き流しを防ぐ。
4. 時間があれば：「先生たちに一言で伝えるなら？」で締める。

## 10. 振り返り（2問）

1. 今日、グラフのどこを根拠にして考えましたか。
2. 友達のグラフや考えと比べて、変わったこと・付け足したことは何ですか。

## 11. 準備・環境

- 授業開始前に、教科書・ノート・筆記用具を机上にそろえさせておく（十文字先生助言・即時実施）。
- 大判方眼シート（目盛り数値なし）…グループ数＋予備／個人ワークシート…児童数分／課題文・表の拡大掲示物／色チョーク（境目の議論用）／マグネット。

## 12. 研究協議で見ていただきたい視点

- 目盛り・向き・並べ方を委ねたことで、児童の工夫や困りが学びの起点になっていたか。
- 「②相談して決めたい場所」が、迷いを表現する装置として機能していたか。
- 教師が復唱・説明・ヒント出しを我慢し、グループに返せていたか（説明で解決する場面を3回以内に）。
- 学びに混ざりにくい児童に、教師ではなく友達がつながる場面があったか。
